# 0.0.5

**Released:** 2026-03-11

## Stable Release

Promoted the 0.0.4-beta.9 runtime and packaging fixes into the stable 0.0.5 line.

### Changes

- Fixed installed-runtime release harness to derive LocalNest paths from active user home instead of machine-specific path
- Hardened installed-runtime validation with MCP stdio handshake regression test and release-sweep coverage
- Switched upgrade skill-sync flow to primary `localnest install skills --force` command while keeping legacy alias

### Upgrade

```bash
npm install -g localnest-mcp@0.0.5
```
