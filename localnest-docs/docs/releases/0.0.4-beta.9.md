## Summary

`0.0.4-beta.9` tightened packaged-skill version reporting and package metadata alignment while keeping the beta.8 runtime baseline intact.

## Key Changes

- fixed bundled skill version reporting so the installed skill shows the actual package version
- made the skill installer use `package.json` as the version source of truth
- added regression coverage to keep bundled skill metadata aligned with package metadata

## Practical Impact

This beta cleaned up user-visible version drift around skill installation without changing the broader retrieval, memory, or runtime contract established in beta.8.

## When to Use This Page

- debugging a machine pinned to `0.0.4-beta.9`
- comparing the frozen beta.9 package against later current-release docs
- checking when bundled skill version reporting was corrected
