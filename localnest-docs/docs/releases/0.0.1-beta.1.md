# 0.0.1-beta.1

Initial beta release notes for `0.0.1-beta.1`, including the first semantic indexing service, hybrid search support, and doctor command for environment validation.

## Summary

This is the first published LocalNest beta documented in the changelog.

## Key Changes

- single-package Node.js layout at the repository root
- `localnest-mcp-doctor` command
- first semantic indexing service
- initial `index_status`, `index_project`, and `search_hybrid` additions
- automatic config migration with backup creation

## Practical Impact

This is the starting point for the modern LocalNest package line documented in this site.
