# 0.0.2-beta.2

Release notes for `0.0.2-beta.2`, introducing canonical `localnest_*` naming, structured response formatting, pagination metadata, and bundled skill distribution.

## Summary

This beta turned LocalNest into a more complete MCP package and distribution flow.

## Key Changes

- canonical `localnest_*` tool names added while legacy aliases remained
- `response_format` support for `json` and `markdown`
- pagination metadata on list-style tools
- bundled skill distribution
- `localnest-mcp-install-skill` command
- broader test coverage across config, search, workspace, and indexing

## Practical Impact

This is the first version where the package and client integration story feels coherent.
