# Plan 09-01: MCP Tool Registration for Graph, Nest, Diary, Ingest, and Dedup

## Objective

Register all 17 new MCP tools covering knowledge graph, nest/branch, graph traversal, agent diary, conversation ingestion, and duplicate detection operations. Follow the exact pattern from memory-store.js.

## Context

- @src/mcp/tools/memory-store.js (pattern to follow)
- @src/services/memory/service.js (facade methods)

## Tasks

### Task 1: Create graph-tools.js registrar
type="auto"

Create `src/mcp/tools/graph-tools.js` exporting `registerGraphTools({ registerJsonTool, schemas, memory })`.

Register 17 tools:
- 7 KG tools (localnest_kg_*)
- 3 Nest tools (localnest_nest_*)
- 2 Graph tools (localnest_graph_*)
- 2 Diary tools (localnest_diary_*)
- 2 Ingest tools (localnest_ingest_*)
- 1 Dedup tool (localnest_memory_check_duplicate)

Each tool uses zod schemas and registerJsonTool with proper annotations.

**Done when:** File exists with all 17 tools registered.

### Task 2: Wire into index.js and register-tools.js
type="auto"

1. Add `export { registerGraphTools } from './graph-tools.js'` to `src/mcp/tools/index.js`
2. Import and call `registerGraphTools` in `src/app/register-tools.js`

**Done when:** registerGraphTools is exported and called during server startup.

## Verification

- All 17 tool names exist in graph-tools.js
- index.js exports registerGraphTools
- register-tools.js calls registerGraphTools
