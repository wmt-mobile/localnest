Implement `backupDatabase(adapter: Adapter, destPath: string): Promise<BackupResult>`:
1. Resolve destPath to absolute path via `path.resolve(destPath)`.
2. Ensure parent directory exists: `fs.mkdirSync(path.dirname(resolvedDest), { recursive: true })`.
3. Sanitize for SQL interpolation: `const safeDest = resolvedDest.replace(/'/g, "''")`.
4. Call `await adapter.exec(\`VACUUM INTO '${safeDest}';\`)`.
5. Open a temporary `new DatabaseSync(resolvedDest)` connection, run `PRAGMA integrity_check` (use `.prepare('PRAGMA integrity_check').all()`), collect the first row's value, close with `.close()`.
6. Return `{ path: resolvedDest, size_bytes: fs.statSync(resolvedDest).size, created_at: new Date().toISOString(), integrity }`.

Implement `restoreDatabase(sourcePath: string, destDbPath: string): Promise<RestoreResult>`:
1. Resolve sourcePath to absolute path.
2. If file does not exist (`!fs.existsSync(resolvedSrc)`), throw `new Error(\`Backup file not found: ${resolvedSrc}\`)`.
3. Open a temporary `new DatabaseSync(resolvedSrc)`, run `PRAGMA integrity_check`, collect integrity value, close.
4. If integrity !== 'ok', throw `new Error(\`Backup integrity check failed: ${integrity}\`)`.
5. `fs.copyFileSync(resolvedSrc, destDbPath)`.
6. Return `{ restored_from: resolvedSrc, db_path: destDbPath, restart_required: true, integrity }`.
